### Soup Defense

A simple top-down strategy game built with Phaser.js.

The player must defend a soup using various ingredients while mold tries to take it over.
Each ingredient plays a role in defense, and the mold gradually spreads across the soup.

![main-screenshot](screenshots\main-screenshot.png)

Project Setup
Development Mode
```bash
npm run dev
```

After launch, the game will be available at:
http://localhost:3000

Tests
```bash
npm run test
```

Tests are implemented using Playwright.
